# Tu Almacén Vecino

Sistema de carrito de compras para tienda en línea con gestión completa de productos.

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/DevsataAlbo/prueba_modulo_04.git
cd prueba_modulo_04
```

## Ejecutar en local

Abre el archivo `index.html` directamente en tu navegador, o usa Live Server en VS Code.

## Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Bootstrap 5.3
- Font Awesome 6.0

## Funcionalidades

- Modal de bienvenida con registro de usuario
- Carousel de productos destacados
- Filtro de productos en tiempo real
- Carrito de compras con cálculo automático de totales
- Cálculo de IVA y despacho
- Tooltips interactivos
- Diseño responsive
